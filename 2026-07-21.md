## New Models & Tools

- **Moonshot AI's Kimi K3** is now live via the Kimi API and apps — a 2.8-trillion-parameter sparse Mixture-of-Experts model billed as the largest open AI model in the world. It natively handles text, image, and video, supports a 1M-token context window, and uses MXFP4 quantization to shrink storage to roughly 1.4TB; full open weights are scheduled for July 27. Pricing is about $3/M input and $15/M output tokens. [llm-stats.com](https://llm-stats.com/ai-news)
- **AWS added a Web Search tool for Bedrock agents**, letting agents pull live web results into answer generation without custom retrieval plumbing. [llm-stats.com](https://llm-stats.com/llm-updates)

## Company & Product Updates

- **The European Commission's binding DMA measures against Google took effect this week**: Google must let users set third-party AI assistants as system defaults (e.g., via a "Hey Google"-style voice trigger) and let rival AI assistants act on-device, with search-data sharing to competitors beginning January 2027 and full Android feature access due by August 2027. Google's Kent Walker publicly objected, citing privacy and security risks. [Digital Markets Act (EC)](https://digital-markets-act.ec.europa.eu/commission-provides-guidance-google-ai-interoperability-android-and-sharing-google-search-data-under-2026-07-16_en)
- **Anthropic's Claude Sonnet 5 is now the default model for all free and paid Claude users**, built on the new "Mythos" architecture and reportedly reaching performance close to Anthropic's larger Opus-class models on reasoning and creativity benchmarks, at lower introductory pricing. [dentro.de](https://dentro.de/ai/news/)
